# HaiDIS: Hospital AI Decision Intelligent System
## Overview
AI-powered decision intelligence system for hospital resource optimisation, reducing waiting time, improving staff allocation, and enhancing operational efficiency using Z.AI GLM.

## Features
- Cost vs Efficiency Analysis
- Real-time Simulation Updates
- Waiting Time Estimation
- AI Recommendations using GLM API

## Technologies Used
- Python (Simulation Engine)
- requests (API communication)
- GLM API (AI recommendations)
- JavaScript (Frontend)

## How to Run
**IN TERMINAL:**\
pip install -r requirements.txt\
python main.py

## Example Output
Cost Change: +1200\
Waiting Time Change: -10 minutes\
Interpretation: Cost increased ⚠️ | Waiting time improved 🚀

## Limitations
- Uses simulated data
- Depends on external API availability
